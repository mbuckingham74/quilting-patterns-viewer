# Quilting Patterns - Feature Roadmap

## Completed Features

- [x] Pattern browsing with pagination
- [x] AI-powered natural language search (Voyage AI + pgvector)
- [x] Pattern sharing with customer rankings
- [x] User favorites and saved searches
- [x] Duplicate pattern detection
- [x] Admin user management
- [x] Bulk pattern upload from ZIP
- [x] Pattern metadata editor with keyword management
- [x] How-To guide for non-technical users
- [x] Error handling with Sentry integration
- [x] Pattern Analytics Dashboard (downloads, views, searches, user activity, top patterns, top searches)

---

## Upcoming Features

### 1. Keyword Management
- Add/edit/delete keywords
- Merge duplicate keywords
- View patterns without any keywords assigned
- Bulk assign keywords to multiple patterns

### 2. Share Analytics
- View all active/expired shares
- See which patterns get the highest customer rankings
- Resend share emails
- Extend expiration dates

### 3. Orphan Pattern Finder
- Find 301 patterns missing thumbnails
- Find patterns missing embeddings (can't AI search)
- Bulk regenerate embeddings for patterns

### 4. Bulk Pattern Operations
- Select multiple patterns with checkboxes
- Bulk add/remove keywords
- Bulk delete patterns
- Export pattern list to CSV

### 5. Favorites Enhancements
- Create named collections/folders
- Share favorite collections with other users
- Export favorites list

### 6. Search History & Suggestions
- Show recent searches
- Auto-suggest based on popular searches
- "Similar to this pattern" button

### 7. Admin Activity Log
- Track who approved users, deleted patterns, etc.
- Simple audit trail

### 8. Bulk User Approval
- Checkbox to select multiple pending users
- "Approve All Selected" button

### 9. Pattern Preview Zoom
- Click to open larger preview modal
- Pan/zoom on large patterns
